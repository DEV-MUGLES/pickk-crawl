## Item Info Crawler

> Zeit now, Cheerio

### Getting Started

Need now 2.0 (Latest) to deploy

```
npm i -g now
```

```shell script
# test locally
$ now dev
# deploy
$ now
```

### Deploy with ZEIT Now

[![Deploy with ZEIT Now](https://zeit.co/button)](https://zeit.co/import/project?template=https://github.com/gywlsp/item-info-crawl)
